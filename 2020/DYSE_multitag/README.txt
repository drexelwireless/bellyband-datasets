Each directory uses the following naming convention:

**bpm**c**sps_**sw
------------------
**bpm - breaths per minute
**c - noise scale (factor used to scale noise using rayleigh fading model)
**sps - samples per second
**sw - length of sliding window in seconds

In each directory is snr_vals.csv. The index of each row corresponds with
the number assigned to each data set, i.e. 00, 01, 02, etc.
